== Description ==

A cool image gallery runs images and videos(youTube and vimeo videos) with responsive design.
More images loads on scroll down using Ajax call so it very cool and lightweight plugin.

Here are the key features:

*   Custom Image and video gallery 
*   Cool popup to see image and videos
*   Due to custom post type user will select category and all post to related that category on front comes as thumbnail
*   on thumbnail click popup will open and user can see image and play the youtube and vimeo video
*   Sort code used for gallery View.more dynamic shortcodes available to show image and videos category wise.
*   gallery data list view with Auto loading based mouse scroll events and image/video are loading via AJAX call which will not load all the post at once.
*   Tested only with the latest version WP 3.5.1 but should work with previous version but not guaranteed.

== Installation ==

1. Upload folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. See the setting for plugin on custom post type WP Cool gallery and select category to add then add post with related category.
4. From post we can select image OR video(support youtube and vimeo videos link) and related category.
5. From setting select shortcode and add on page or post.
